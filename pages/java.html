Ümumi Məlumat

Back-end developer veb proqramın və ya vebsaytın server tərəfini qurmaq və saxlamaqda ixtisaslaşan proqram mühəndisidir. Onlar proqrama güc verən kodun və məntiqin işlənib hazırlanmasına, həmçinin onun təhlükəsizliyini, genişlənməsinə və etibarlılığına cavabdehdirlər.

Back-end tərtibatçıları adətən PHP, Python, Ruby, Java, Node.js kimi dillər və texnologiyalar və MySQL, MongoDB və Oracle kimi verilənlər bazaları ilə işləyirlər. Onlar proqramın arxitekturasını və dizaynını yaradır, server tərəfindəki kodu qurur və saxlayır və tətbiqi digər sistem və xidmətlərlə inteqrasiya edirlər.
